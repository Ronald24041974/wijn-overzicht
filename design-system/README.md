# Wijnoverzicht – Design System

Dit is het design system voor de **Wijnoverzicht** app. Gebruik deze map als bron voor visuele beslissingen, nieuw te bouwen schermen of aanpassingen.

## Inhoud

| Bestand            | Beschrijving |
|--------------------|--------------|
| `tokens.css`       | Alle design tokens als CSS-variabelen (kleuren, typografie, vormen, schaduwen) |
| `components.md`    | Overzicht van alle UI-componenten met HTML-voorbeelden en gebruik |

## Kernprincipes

1. **Verfijnd & rustig** — subtiele schaduwen, weinig kleuren, veel witruimte
2. **Groen als merk** — donkergroen (`#213f39`) is de primaire kleur voor alle acties
3. **Goud als accent** — goud (`#f2c36b`) markeert actieve/geselecteerde elementen
4. **Mobiel eerst** — app werkt als PWA op iPhone, mobiele lay-out is leidend
5. **Duidelijke hiërarchie** — grote KPI-waarden, kleine labels, subtiele muted tekst
